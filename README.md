# ArtıGıda Platformu 🍎

ArtıGıda, restoran, fırın ve marketlerin gün sonunda satılmayan ancak taze olan gıdalarını çöpe atmak yerine, ihtiyaç sahibi öğrencilere ve yerel halka ulaştırmasını sağlayan bir köprü platformdur.

Bu proje, **Future Talent 301 YZ ile Ürün Geliştirme Bootcamp** kapsamında geliştirilmekte olup, yapay zeka destekli bir sosyal dayanışma ağı sunmayı amaçlamaktadır.
